# 📝 To-Do List Project

## 📌 Overview
The To-Do List Project is a simple command-line application developed in Python. It helps users organize their daily tasks by allowing them to add, view, and delete tasks through an interactive menu-driven interface.

## ✨ Features
- Add new tasks
- View all saved tasks
- Delete completed tasks
- User-friendly menu interface
- Input validation for better user experience

## 🛠️ Technologies Used
- Python 3

## 🚀 How to Run
1. Open the project in Visual Studio Code.
2. Run the Python file.
3. Use the menu options to add, view, or delete tasks.

## 🎯 Learning Objectives
This project was developed to strengthen the following Python concepts:
- Functions
- Lists
- Loops
- Conditional Statements
- Exception Handling
